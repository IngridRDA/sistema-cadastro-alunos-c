Projeto desenvolvido para a disciplina de **Algoritmos e Programação** da Universidade Unigranrio.

---

## 📖 Sobre o projeto

Este sistema foi desenvolvido utilizando a linguagem **C** com o objetivo de aplicar conceitos fundamentais de programação estruturada.

O programa realiza o gerenciamento de alunos através de um menu interativo no terminal, utilizando **structs**, **vetores**, **funções** e **validações de dados**.

---

## ✨ Funcionalidades

- Cadastro de alunos
- Listagem de registros
- Busca por matrícula
- Atualização da nota final
- Validação de matrícula duplicada
- Validação de notas
- Organização modular do código

---

## 🛠 Tecnologias

- Linguagem C
- Struct
- Vetores
- Modularização
- Programação Estruturada

---

## 📂 Estrutura

```
Aluno
├── Nome

├── Matrícula
└── Nota Final
```

---

## 🚀 Como executar

Compilar:

```bash
gcc src/cadastro_alunos.c -o cadastro
```

Executar:

Linux/macOS

```bash
./cadastro
```

Windows

```bash
cadastro.exe
```
